# Tag
Tags are used to categorize, identify and organize content.


## Properties

<Props of={BaoTagComponent} />

## Type

* `'positive'` is a green tag
* `'negative'` is a red tag
* `'alert'` is a yellow tag
* `'info'` is a blue tag
* `'neutral'` is a grey tag

## Variant

* `'light'` tag's color is light
* `'strong'` tag's color is bold
